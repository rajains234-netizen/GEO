## Full System Build

The complete system build process includes the following steps:
1. **Clone the repository:**
   ```bash
   git clone https://github.com/rajains234-netizen/geo.git
   ```

2. **Install dependencies:**
   Navigate to the project directory and install all required dependencies:
   ```bash
   cd geo
   npm install
   ```

3. **Build the project:**
   Build the project using the provided build scripts:
   ```bash
   npm run build
   ```

4. **Run the project:**
   Start the application:
   ```bash
   npm start
   ```

## System Architecture

The architecture of the system is designed to handle various components efficiently, including:
- **Frontend:** Built using React.js for a responsive user interface.
- **Backend:** Node.js and Express for handling API requests and serving the frontend.
- **Database:** MongoDB as the primary data storage solution.
